# AI Resume & Interview Assistant

## Objective
To help job seekers generate professional resumes, interview questions, and job application emails using AI.

## Tool Used
- ChatGPT

## Problem Solved
Many freshers struggle with resume writing and interview preparation. This project uses AI to generate structured and professional content.

## Features
- Resume summary generation
- Interview Q&A creation
- Job application email drafting
- Resume improvement suggestions

## Skills Demonstrated
- Prompt engineering
- AI content generation
- Career-focused problem solving
- Professional documentation

## Outcome
Created an AI-powered assistant that improves job readiness and confidence for candidates.
